Detailed Description

This header file defines the Inet::RawEndPoint class, where the CHIP Inet Layer encapsulates methods for interacting with IP network endpoints (SOCK_RAW sockets on Linux and BSD-derived systems) or LwIP raw protocol control blocks, as the system is configured accordingly.
